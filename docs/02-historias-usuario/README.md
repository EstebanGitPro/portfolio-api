# Historias de Usuario

Cada HU sigue este formato:

```markdown
# HU-XXX: Título descriptivo

**Como** [rol]  
**Quiero** [acción]  
**Para** [beneficio]

**Criterios de aceptación:**
- [ ] Criterio 1
- [ ] Criterio 2

**Notas técnicas:**
- Endpoint: `GET /api/...`
- ¿Requiere auth? Sí / No
```
